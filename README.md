# Gestión de Vacaciones mediante Chatbot

## Descripción

Trabajo Práctico Integrador de Organización Empresarial.

El proyecto simula un chatbot para la gestión de solicitudes de vacaciones utilizando Python y una plantilla Excel como base de datos.

## Funcionalidades

- Validación de legajo.
- Consulta de saldo de vacaciones.
- Validación de disponibilidad de fechas.
- Registro de solicitudes.
- Manejo de errores de entrada.
- Gestión de estados del proceso.

## Tecnologías utilizadas

- Python 3
- OpenPyXL
- Excel
- Bizagi Modeler

## Archivos del proyecto

- chatbot_gestion_vacaciones.py
- DB_Gestion_Vacaciones.xlsx
- TPI_Gestion_Vacaciones_Alexis_Torrez.pdf

## Ejecución

Ejecutar el archivo:

```bash
python chatbot_gestion_vacaciones.py
```
